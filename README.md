# 🔐 Crypto Vault

**Learn cryptography basics hands-on** — encrypt messages, hash passwords, generate keys and verify digital signatures.

## Quick Start

Open `index.html` in any browser — no install, no server.

## Files

| File | Description |
|------|-------------|
| `index.html` | Main app |
| `script.js` | Cryptography logic (Web Crypto API) |
| `style.css` | UI styles |
| `icon.svg` | App icon |
| `logo.svg` | App logo |

## Features

- Symmetric & asymmetric encryption demos
- Password hashing visualization
- Key generation and export
- Digital signature creation & verification
- 100% client-side, zero dependencies

## Tech

HTML · CSS · JavaScript · Web Crypto API
